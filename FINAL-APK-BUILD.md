# APK Build Guide

## Quick Build Process

1. **Download this project** from Replit
2. **Install Android Studio** (with Android SDK)
3. **Run build command**:
   ```bash
   cd android
   ./gradlew assembleDebug
   ```
4. **Get APK**: `android/app/build/outputs/apk/debug/app-debug.apk`

## Requirements
- Android Studio with SDK
- Java 17+

## What You Get
